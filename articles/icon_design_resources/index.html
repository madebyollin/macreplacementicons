Apple:

  • macOS Icon Design Guidelines
    Apple's icon guidelines contain a lot of the basic information about how to make a good icon.
  • Application Icons [Need a Mac]
    To view an Application's icon in full resolution, right click on the app in Finder and select "Show Package Contents". Then open the Contents/Resources folder and look around for a file with the extension .icns. That's the icon file. You can use Quicklook [Spacebar] to view the full size only, or open it in Preview to see all of the different sizes.
